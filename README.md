## What is BitcoinBT?

**BitcoinBT (BTCBT)** is a full-node implementation derived from **Bitcoin Core v26**.

The chain is designed as a **Bitcoin hard fork**, inheriting the complete Bitcoin blockchain history up to **block height 903,844** (July 3, 2025).  
From **block 903,845 onward**, BitcoinBT applies its own consensus rules.

BitcoinBT does **not** replace Bitcoin and does **not** interact with the Bitcoin main network.

**Important Notes**
- Bitcoin holders do **not** automatically receive BTCBT
- BTCBT is mined independently after the fork point

## Special Block Reward (Block 903,845)

At the fork activation point (**block 903,845**), BitcoinBT includes a **one-time special block reward of 2,000,000 BTCBT**.

This reward is:

- Generated by **mining**, not pre-issued
- Created as part of the **first BitcoinBT coinbase block**
- Not derived from Bitcoin balances, snapshots, or airdrops
- Fully visible and verifiable on-chain

This reward exists to ensure **long-term operational viability** of the BitcoinBT network.
